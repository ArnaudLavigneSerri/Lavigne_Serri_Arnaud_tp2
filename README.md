# als_tp2

Developed with Unreal Engine 5

Dans le premier niveau le joueur va devoir trouvé un staff de rapetissement pour changer les pois d’une balanes pour atteindre la porte. Dans le deuxième niveau le joueur va devoir trouver un staff de propulsion pour propulser un objet lourd pour détruire un mur puis il va devoir trouvé le staff ultime pour ouvrir un portaille pour réussir et s’échappé.
